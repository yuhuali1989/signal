## 关键技术创新

### 1. 未来帧回溯 (Hindsight Labeling)

核心洞察：**标注当前帧时，可以利用未来已经发生的事实**。

例如：当前帧一个物体被遮挡了 80%，人类标注者很难判断其类别和大小。但 3 秒后遮挡消失，物体完全可见。利用未来帧的信息可以准确标注当前帧。

```python
# 回溯标注伪代码
def hindsight_label(frame_t, future_frames):
    """利用未来帧回溯标注当前帧"""
    
    # 1. 在未来帧中检测完全可见的物体
    detections = []
    for frame in future_frames:  # t+1 到 t+10s
        dets = detector.detect(frame)
        detections.extend(dets)
    
    # 2. 跟踪回当前帧
    tracks = tracker.backward_track(detections, to_frame=frame_t)
    
    # 3. 多帧投票取共识
    labels = temporal_voting(tracks, min_votes=5)
    
    # 4. 投影到当前帧验证
    for label in labels:
        reprojection_error = project_and_verify(label, frame_t)
        label.confidence = 1.0 - reprojection_error / max_error
    
    return labels
```

### 2. 三层质量保证

| 层级 | 方法 | 目标 |

### 3. 与世界模型的协同

4D 自动标注不仅用于训练 VLA 模型，还用于训练世界模型：


这形成了**标注→训练→生成→标注**的二级飞轮。
