### 1.2 Engram 记忆架构

V4 的一个重大创新是引入 **Engram Memory Architecture**——一种可学习的长期记忆机制：

```
传统 Transformer:
  输入 → Attention → FFN → 输出
  (每次推理都从零开始)

Engram 记忆 Transformer:
  输入 → Attention → FFN → 输出
              ↕
        Engram Memory Bank
  (跨推理保持的可学习记忆向量)
```

Engram 的核心思路是在 Transformer 层之间插入一组**持久记忆向量（Persistent Memory Vectors）**，这些向量在推理过程中被读写，并在多次调用间保持状态。这让模型具备了类似"工作记忆"的能力。
