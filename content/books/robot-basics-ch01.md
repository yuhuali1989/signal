# 第 1 章：材料基础与结构设计

> **学习目标**：能根据机器人场景（桌面级/移动/工业）选择合适结构材料，理解关键力学参数，会用 3D 打印快速出原型。

---

## 1.1 为什么材料选型至关重要

机器人的性能上限，很大程度上由材料决定：

- **自重**决定载重比（有效载荷 / 自重），自重越轻运动效率越高
- **刚度**决定位置精度（刚度不足 → 末端振动 → 视觉/力控误差）
- **成本**决定产品化可能性

一个典型的矛盾：7 轴机械臂如果全用钢材，自重可能超过 80kg，有效载重比 < 5%；换成碳纤维管 + 铝合金关节，自重可降至 12kg，载重比提升至 30%。

## 1.2 主要结构材料对比

### 铝合金

机器人设计中使用最广泛的金属材料。

| 牌号 | 屈服强度 | 密度 | 适用场景 | 加工性 | 参考价 |
|------|---------|------|---------|--------|--------|
| 6061-T6 | 276 MPa | 2.7 g/cm³ | 机身骨架、连杆 | ★★★★★ | ¥30/kg |
| 7075-T6 | 503 MPa | 2.81 g/cm³ | 高应力关节、起落架 | ★★★★☆ | ¥80/kg |

**选用原则**：
- 结构件首选 6061-T6：成本低，CNC 加工容易，阳极氧化后耐腐蚀
- 高载荷关节用 7075-T6：但焊接性差，通常用螺栓连接

```
CNC 加工典型公差：
- 孔径: H7/h6 配合（间隙 ≤ 0.02mm）
- 平面度: ≤ 0.05mm
- 表面粗糙度 Ra: ≤ 1.6μm（轴承座面）
```

### 碳纤维复合材料（CFRP）

无人机、轻量化机械臂的首选轻量化材料。

**关键参数**：
- T300 标准模量碳纤维布：E = 230 GPa，σ_ult = 3500 MPa，ρ = 1.76 g/cm³
- 比强度（UTS/ρ）= 1989 MPa·cm³/g —— 是铝合金的 **6.4 倍**，钢的 **10 倍**

**常用规格与价格**：

|------|------|--------|
| Ø16×14mm 碳管 | 六旋翼机臂 | ¥25/根 |

| 1.5mm 碳板 | FPV 机架 | ¥35/片 |

**注意**：碳纤维导电（碳/铝接触会发生电偶腐蚀），接触面需涂环氧胶或垫绝缘垫片。

## 1.3 结构设计基本原则

### 力矩与截面设计

机械臂最常见的失效模式是**弯曲变形**，而非拉伸断裂。

对于悬臂梁（关节连杆），最大挠度：

```
δ = F·L³ / (3·E·I)

其中：
  F = 末端载荷 + 自重等效力 (N)
  L = 连杆长度 (m)
  E = 弹性模量 (Pa)
  I = 截面二次矩 (m⁴)

圆管截面: I = π(D⁴-d⁴)/64
矩形截面: I = bh³/12
```

**设计目标**：在给定载荷下，δ/L < 0.001（千分之一挠度比）

### 螺栓连接

机器人关节大量使用内六角螺栓（ISO 4762），选型要点：

- **强度等级**：8.8 级（σy = 640 MPa）足够大多数场景；12.9 级用于高强度连接
- **预紧力矩**：M6 × 8.8 级 → 预紧力矩 10 N·m；M3 → 0.9 N·m
- **防松**：振动环境必须涂螺纹胶（Loctite 243，可拆卸）或用弹垫

```python
# 螺栓预紧力计算
import math

def bolt_torque(d_mm, grade, mu=0.15):
    """
    d_mm: 螺栓公称直径 (mm)
    grade: 强度等级 (如 8.8)
    mu: 摩擦系数 (通常 0.12~0.18)
    返回: 推荐预紧力矩 (N·m)
    """
    # 屈服强度 (MPa)
    grade_map = {4.6:240, 5.8:400, 8.8:640, 10.9:900, 12.9:1080}
    sigma_y = grade_map.get(grade, 640)

    # 螺纹中径 (近似 d × 0.9)
    d2 = d_mm * 0.9 * 1e-3   # m
    d_m = d_mm * 1e-3

    # 预紧力：F = 0.7 × σy × A_stress
    A_stress = math.pi / 4 * (0.9 * d_mm * 1e-3)**2
    F = 0.7 * sigma_y * 1e6 * A_stress

    # 拧紧力矩
    T = F * mu * d_m * 0.5 * 1.3   # 含螺纹+支承面摩擦
    return round(T, 2)

for d in [3, 4, 5, 6, 8, 10]:
    print(f"M{d} 8.8级 预紧力矩: {bolt_torque(d, 8.8)} N·m")
```

## 1.5 本章小结

- **铝合金**是机器人结构的万能选择：6061 用于大多数场景，7075 用于高应力
- **碳纤维**能减重 40%+，但成本高且加工需专用刀具，适合轻量化优先的场景
- **3D 打印**是快速迭代神器：PA12 SLS 打印的零件可直接用于功能验证
- **结构刚度**比强度更关键：截面惯性矩设计优先，避免悬臂过长
